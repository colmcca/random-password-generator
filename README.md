# Random Password Generator

## Description

The goal of this application is to generate a random password based on the prompts asked. You choose a length between 8 and 128 characters and then are able to choose if you want lowercase letters, uppercase letters, symbols, and numbers included in your password. Once you finish answering the prompts, the password will generate.
